# Othalo - Sustainable Housing Solutions

Othalo is a patented technology platform that transforms waste plastic into high-quality, structural wall panels. This project is a professional web application designed to showcase Othalo's innovative approach to solving the global plastic waste and affordable housing crises.

## 🚀 Overview

The Othalo platform addresses two major global challenges simultaneously:
1. **Plastic Waste**: Transforming environmental liabilities into structural assets.
2. **Affordable Housing**: Delivering homes with speed, scale, and high thermal efficiency.

Each 2-bedroom housing unit diverts approximately **8 tonnes** of plastic from the environment.

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **i18n**: [react-i18next](https://react.i18next.com/)
- **State Management**: [React Query](https://tanstack.com/query/latest)

## ✨ Key Features

- **Dynamic Video Hero**: High-impact video background on the landing page.
- **Project Quote System**: Interactive modal for requesting regional project quotes.
- **Multi-language Support**: Supports English, Norwegian, French, Spanish, Portuguese, Arabic, and Russian.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Structural Integrity**: Endorsed by UN-HABITAT and tested to international building standards.

## 🚀 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run locally**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 🌍 Impact

Othalo panels are 7x stronger than concrete, 3x more thermally efficient, and are designed to last for over 60 years. By using 100% recycled HDPE, Othalo is building a sustainable future, one panel at a time.
